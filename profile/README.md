## Architecture

Celio-Link consists of three main components:

### Celio-Server
Hosts and manages online sessions between players.

### Celio-Client
Browser-based frontend used to create and join sessions.

### Celio-Firmware
Firmware that enables communication between the Game Boy Advance and the browser.
